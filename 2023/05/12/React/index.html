React 简介

是什么

  1. 用于动态构建用户界面的 JavaScript 库（只关注于视图）
  2. 由 Facebook 开源：Facebook开发，起初由 Facebook 的软件工程师 Jordan Walke 创建，于 2011 年部署于 Facebook 的 newsfeed ，随后在 2012 年部署于 Instagram，2013年 5 月宣布开源。

==jsx：解决虚拟 DOM 创建繁琐的问题==

JavaScript 的问题

  • 原生 JavaScript 操作 DOM 繁琐、效率低（DOM-API 操作 UI )
  • 使用 JavaScript 直接操作 DOM，浏览器会进行大量的重绘重排
  • 原生 JavaScript 没有组件化编码方案，代码复用率低

React 的特点

  1. 采用声明式编码、组件化编码提高开发效率以及组件复用率

  2. 在 React Native 中可以使用 React 语法编写原生应用

  3. 高效（优秀的 Diffing 算法），使用虚拟DOM + Diffing 算法，尽量减少与真实 DOM 的交互

Download the React DevTools for a better development experience: https://fb.me/react-devtools

Be sure to precompile your scripts for production - https://babeljs.io/docs/setup/
