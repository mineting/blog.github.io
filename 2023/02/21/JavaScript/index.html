JavaScript 概述

权威地址（MDN）：https://developer.mozilla.org/zh-CN/docs/web/JavaScript

JavaScript：运行在客户端（浏览器）的编程语言。

JavaScript（JS）是一种具有函数优先特性的轻量级、解释型或者说即时编译型的编程语言。虽然作为 Web 页面中的脚本语言被人所熟知，但是它也被用到了很多非浏览器环境中，例如 Node.js、Apache CouchDB、Adobe Acrobat 等。进一步说，JavaScript 是一种基于原型、多范式、单线程的动态语言，并且支持面向对象、命令式和声明式（如函数式编程）风格。

JavaScript 的作用

  • 网页特效
  • 表单验证
  • 数据交互
  • 服务端编程（Node）

JavaScript 的组成：

  • ECMAScript：规定了基础的语法

  • WebApis

    • DOM：操作文档，主要对页面文档进行移动、修改大小、删除等
    • BOM：操作浏览器，如页面弹窗、检测窗口进度、存储数据到浏览器
